# Node Wrangler Colorful (炫彩节点牧人) ✨

一款基于 Blender 官方 Node Wrangler 深度定制的增强版插件。它不仅继承了原版强大的节点管理能力，还引入了令人心动的流光视觉特效和全动态中英双语支持。

## 🌟 核心特色 (Core Features)

### 🌈 动态流光视觉 (Visual Aesthetics)
- **彩虹连线**：连接选中的节点时，会自动显示具有流动感的彩虹线条，节点逻辑一目了然喵。
- **动态边框**：选中节点时拥有炫酷的流光边框特效。
- **六大流光主题**：
  - **经典彩虹**：全光谱流动脉冲。
  - **赛博霓虹**：青色与品红的高频数据流。
  - **魔法流金**：温暖涌动的岩浆质感。
  - **幽能矩阵**：深邃的《黑客帝国》代码流。
  - **极地冰晶**：清凉柔和的浅蓝水流。
  - **少女粉 (New!)**：高饱和度的粉嫩心动流光。

### 🌍 完美中英双语 (Full i18n Support)
- **动态切换**：无需重启，随插件语言设置实时切换界面标签、菜单标题及操作提示。
- **深度汉化**：针对节点操作提供了更符合中文习惯的翻译条目喵。

## ⌨️ 热键秘籍 (Shortcuts)

### 快速连接与混合 (Quick Link & Mix)
- `Alt + 右键拖拽`：**快速连接** (Lazy Connect)，智能识别端口喵。
- `Alt + Shift + 右键拖拽`：**带菜单连接**，手动选择端口。
- `Ctrl + Shift + 右键拖拽`：**快速混合** (Lazy Mix)，交互式添加混合节点。
- `Alt + S`：**交换链接**，支持两个节点的输出交换或单个节点的输入交换。

### 节点合并 (Merge Nodes)
- `Ctrl + 0 (数字键)`：自动混合 (Auto Mix)。
- `Ctrl + + (加号)`：相加 (Add)。
- `Ctrl + * (乘号)`：相乘 (Multiply)。
- `Ctrl + / (除号)`：相除 (Divide)。
- `Ctrl + Alt + 0`：Alpha Over 合并。

### 材质设置 (Texture Setup)
- `Ctrl + T`：**一键纹理设置**，自动添加纹理节点、坐标和映射。
- `Ctrl + Shift + T`：**原理化设置**，批量加载并自动识别连接贴图（基础色、法线、粗糙度等）。

### 整理与清理 (Cleaning & Alignment)
- `Alt + =`：**对齐节点**，将选中的节点整齐排列。
- `Alt + X`：**删除未使用节点**，保持节点树清爽。
- `Shift + P`：将选中节点塞进**框节点**。
- `Backspace`：**重置节点**，恢复默认设置但保持连线喵。

### 快捷菜单 (Menus)
- `Shift + W`：**主菜单**，访问所有 Node Wrangler 功能。
- `/` 或 `小键盘 /`：**路由点菜单** (Reroutes Menu)。
- `\`：**活动项链接到选中项菜单**。
- `Ctrl + C`：**复制到选中项菜单**。

## ⚙️ 插件设置 (Preferences)
在插件首选项中，您可以：
- 自由切换 **Language (语言)**。
- 选择您喜欢的 **流光主题 (Visual Theme)**。
- 调节 **彩虹线增粗** 倍率喵。
- 开启 **性能模式**：当选中超过 5 个节点时自动降低渲染消耗，保证操作流畅。

---

## 🛠️ 安装说明 (Installation)
1. 下载 `Node_Wrangler_Colorful.zip`。
2. 在 Blender 中前往 `编辑 -> 偏好设置 -> 插件 -> 安装`。
3. **重要**：如果已启用官方原版 `Node Wrangler`，请务必先将其禁用，以避免功能冲突喵！
